# Plans

Use this page to find planning material before making feature-sized changes.

## Execution Plans

- Plan instructions: `exec-plans/README.md`
- Active plans: `exec-plans/active/`
- Completed plans: `exec-plans/completed/`
- Technical debt tracker: `exec-plans/tech-debt-tracker.md`

Create an active execution plan before implementing changes that affect user workflows,
public contracts, or multiple layers. Keep small bug fixes and narrow refactors out of
the planning system unless the work becomes cross-cutting.
