# Planned Execution Plans

Use this folder for future FPVS Studio plans that are worth preserving but are not yet
active implementation work.

Move a plan from `planned/` to `active/` when:

- the user has approved the work for implementation
- the plan has clear boundaries and verification
- the next agent should treat it as the current task context

Keep speculative ideas out of this folder unless they are concrete enough to help future
planning.
