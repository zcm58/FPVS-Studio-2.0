# Archived References

This folder preserves historical setup prompts, scaffold plans, and audit notes that are
not current source-of-truth docs.

Use these files only when researching why the repo was originally shaped a certain way.
Do not start current implementation work from this folder. Current task context should
start from `../../../AGENTS.md`, `../../../ARCHITECTURE.md`, and `../../index.md`.

## Archived Files

- `CODEX_INITIAL_PROMPT.md`: initial app/repo prompt context.
- `REPO_SCAFFOLD_PLAN.md`: original scaffold plan.
- `PHASE4_AUDIT.md`: historical Phase 4 audit notes.
- `PY310_COMPAT_AUDIT.md`: historical Python 3.10 compatibility audit notes.
