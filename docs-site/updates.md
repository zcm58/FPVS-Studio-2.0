# Update FPVS Studio

FPVS Studio can check GitHub Releases for newer installers.

## In-App Update Check

Use:

```text
File > Check for Updates
```

If a newer version is available, the update dialog shows the current version, latest
version, a short release note summary, and a link to full release notes.

## Manual Update

You can also download the latest installer from the
[GitHub Releases page](https://github.com/zcm58/FPVS-Studio-2.0/releases/latest).

Run the new installer over the existing installation. User projects remain in the FPVS
Studio Root Folder, not in the application install directory.

## Beta Release Notes

The current beta is Windows-focused and BioSemi-oriented. Review release notes before
updating a lab machine used for active data collection.
