# Update FPVS Studio

FPVS Studio can check GitHub Releases for newer installers each time you launch the app.

## In-App Update Check

Use:

```text
File > Check for Updates
```

If a newer version is available, you'll be notified to download the latest version. 

## Manual Update

You can also download the latest installer from the
[GitHub Releases page](https://github.com/zcm58/FPVS-Studio-2.0/releases/latest).

Run the new installer over the existing installation.
