# Run A Session

Use FPVS Studio to reopen a saved project and launch participant sessions with minimal
setup.

## Launch

1. Open FPVS Studio.
2. Choose **Open Projects** and select the experiment.
3. Choose **Launch Experiment**.
4. Enter the participant number when prompted.
5. Follow the fullscreen prompts.

Playback uses `Space` for start and continue screens.

## Outputs

Detailed run outputs are saved inside the project folder under:

```text
runs\
```

Project-level run history is saved under:

```text
logs\
```

Keep these folders with the project when archiving or transferring study data.

## Before Each Participant

Confirm that:

- the intended project is open
- the participant number is correct
- the EEG recording setup is ready
- the display and response setup match the experiment plan
