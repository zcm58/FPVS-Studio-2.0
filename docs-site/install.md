# Install

## Requirements

- Windows 11.
- BioSemi. 

## Install The App

1. Open the [latest GitHub Release](https://github.com/zcm58/FPVS-Studio-2.0/releases/latest).
2. Download the `FPVS-Studio-Setup-*.exe` installer.
3. Run the installer. 
4. Launch FPVS Studio from the Start Menu or Desktop shortcut.

Note: Windows Smart Screen may flag the .exe as potentially malicious as it's coming from 
an unsigned developer. You'll need to bypass those warnings in order to install FPVS Studio for the first time.

## First Launch

On first launch, FPVS Studio asks you to choose an **FPVS Studio Root Folder**. This is
where all of your future FPVS Experiment folders and their associated logs will be stored. 

Choose a stable local folder such as:

```text
C:\FPVS Studio
```
