# Install

## Requirements

- Windows 11.
- A normal user account that can install desktop applications.
- A local folder where FPVS Studio can store experiments.
- BioSemi-compatible lab workflow for the current beta release.

Avoid placing the FPVS Studio root folder inside OneDrive, Dropbox, Google Drive, or
another synced folder. Sync tools can lock or partially update files while an experiment
is being edited or run.

## Install The App

1. Open the [latest GitHub Release](https://github.com/zcm58/FPVS-Studio-2.0/releases/latest).
2. Download the `FPVS-Studio-Setup-*.exe` installer.
3. Run the installer.
4. Launch FPVS Studio from the Start Menu or Desktop shortcut.

## First Launch

On first launch, FPVS Studio asks you to choose an **FPVS Studio Root Folder**. This is
where future experiment projects, reusable templates, logs, and run outputs are stored.

Choose a stable local folder such as:

```text
C:\FPVS Studio
```

Do not choose the application install folder. The root folder is for your lab projects,
not for installed program files.
