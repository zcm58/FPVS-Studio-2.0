# FPVS Studio

FPVS Studio is a Windows desktop package designed to streamline the creation and running ofFast Periodic Visual
Stimulation (FPVS) experiments. 

The built in project setup wizard guides users through the selection of base and oddball images, 
sets up a fixation cross task, and randomizes the order in which conditions are displayed to the participant
each run. 

The current release is in Beta an is compatible with Windows 11 machines and BioSemi systems.

Future updates will bring compatibility to other EEG filetypes. 


## Start Here

- [Install FPVS Studio](install.md) on a Windows lab machine.
- [Create your first project](quickstart.md) with the guided setup wizard.
- [Prepare image folders](prepare-images.md) before assigning stimuli.
- [Run a participant session](run-session.md) and find saved outputs.
