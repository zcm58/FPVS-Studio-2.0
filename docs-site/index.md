# FPVS Studio

FPVS Studio is a Windows desktop app for creating and running Fast Periodic Visual
Stimulation experiments without writing code. It guides lab users through project setup,
stimulus folders, display timing, fixation behavior, participant launch, and run output.

The current beta release is intended for Windows 11 lab machines and BioSemi workflows.
Future releases may expand supported systems and output formats.

## Start Here

- [Install FPVS Studio](install.md) on a Windows lab machine.
- [Create your first project](quickstart.md) with the guided setup wizard.
- [Prepare image folders](prepare-images.md) before assigning stimuli.
- [Run a participant session](run-session.md) and find saved outputs.

## Download

Download the current installer from the
[FPVS Studio GitHub Releases page](https://github.com/zcm58/FPVS-Studio-2.0/releases/latest).

You do not need Python, PyCharm, Git, or the source repository to use the installed app.
